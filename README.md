# AGH Imperative Programming (C)

A collection of foundational C programs developed during the Imperative Programming course at AGH University of Science and Technology. This repository serves as a practical log of low-level programming tasks, focusing on standard I/O, array manipulation, and algorithmic logic.

## 📂 Current Implementations

### 1. Vector Operations Engine
A console-based utility for 1D array (vector) mathematics.
* **Features:** Array addition, scalar multiplication, and dot product calculation.
* **Generators:** Includes custom implementations of NumPy-like `linspace` and `range` functions in pure C.
* **Concepts Demonstrated:** Passing arrays to functions, fixed-size memory buffers, `switch` statement routing, and floating-point arithmetic.

### 2. ASCII Shape Generator
A terminal graphics generator using nested iterative logic.
* **Features:** Renders complex, stacked triangular shapes based on dynamic user height inputs.
* **Concepts Demonstrated:** Nested `for` loops, standard I/O stream manipulation, and algorithm flow control.

## 🛠 Tech Stack
* **Language:** C (C11 standard)
* **Libraries:** strictly `<stdio.h>` for standard input/output.
